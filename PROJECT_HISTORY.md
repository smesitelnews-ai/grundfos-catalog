# История проекта Grundfos Catalog

- **07.07.2026 18:20:** 
  - Создан репозиторий GitHub `grundfos-catalog`.
  - Инициализирован проект Next.js с Tailwind CSS.
  - Написан Python-скрипт (`build_db.py`) для парсинга исходных данных об артикулах и ценах в магазинах. Скрипт генерирует `public/products.json` с высчетом минимальной цены и формированием ссылок поиска по каждому магазину.
  - Сгенерировано красивое изображение насоса через ИИ для заглушек.
  - Реализован премиальный пользовательский интерфейс (glassmorphism) для Главной страницы и Карточек товаров. В карточках выделяется минимальная цена и наличие товара.
  - Подключены иконки `lucide-react`.

- **08.07.2026 00:20:**
  - Реализовано скачивание ОРИГИНАЛЬНЫХ реальных фотографий насосов для всех 24 моделей.
  - Разработаны и успешно запущены два скрипта парсинга:
    - Скрипт `obadis_scraper.py` извлек 13 изображений с европейского Magento-каталога `obadis.com` с использованием парсинга OpenGraph-метаданных.
    - Скрипт `yandex_images_downloader.py` извлек оставшиеся 11 изображений из выдачи Яндекс Картинок через автоматизацию локального Google Chrome с помощью Playwright (успешно обойдена капча и защита от ботов).
  - Все временные проверочные скрипты и ошибочные файлы картинок (логотипы Kermi) были автоматически очищены.
  - База данных `public/products.json` полностью обновлена, привязаны реальные локальные файлы изображений (`/images/pumps/{article}.jpg`).
  - Успешно выполнена продакшн-сборка проекта (`npm run build`), все 24 страницы товаров генерируются статически без ошибок.
