## 2026-07-08 22:08 (МСК)
- Критическое архитектурное изменение: Запросы к Ozon API переведены с Next.js бэкенда (API routes) на клиентскую часть (Client-Side Fetch с поддержкой CORS).
- Причина: серверы Vercel (США/Европа) блокируются файрволом Ozon по IP, что приводило к 'Connect Timeout 10000ms'.
- Исправлен запрос характеристик: Ozon обновил API и удалил v3/products/info/attributes, теперь используется v4/product/info/attributes.
